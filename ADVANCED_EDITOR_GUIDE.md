# 🎨 Advanced 3D Editor - Blender-like Features Guide

## Overview

The Advanced 3D Editor brings **Blender-like functionality** to your browser! Create professional 3D models with advanced tools and pre-loaded sample models.

## ✨ Blender-like Features

### 🎯 Modes
- **Object Mode**: Select and transform objects
- **Edit Mode**: Edit geometry (coming soon)
- **Sculpt Mode**: Sculpting tools (coming soon)

### 🔧 Operations (Blender-like)

1. **Extrude**: Extend selected geometry
2. **Bevel**: Add beveled edges
3. **Subdivide**: Increase geometry detail
4. **Mirror**: Create mirrored copies
5. **Duplicate**: Copy selected objects

## 🚀 How to Use

### Step 1: Access Editor
- Go to **"🎨 Advanced Editor"** in navigation
- Or visit: `http://localhost:5173/advanced-editor`

### Step 2: Add Objects

#### Add Primitives:
1. Click **"+ Add Primitive"**
2. Choose a shape
3. Object appears in scene

#### Load Sample Models:
1. Click **"📦 Load Sample Models"**
2. Select a model (Home, Car, Truck)
3. Model loads into scene

### Step 3: Edit Objects

1. **Select**: Click on object
2. **Transform**: Use Properties panel
   - Position (X, Y, Z)
   - Scale (X, Y, Z)
   - Rotation (degrees)

### Step 4: Blender Operations

1. Select an object
2. Use operations:
   - **Extrude**: Extends geometry
   - **Bevel**: Adds beveled edges
   - **Subdivide**: Increases detail
   - **Mirror**: Creates mirror copy
   - **Duplicate**: Copies object

### Step 5: Customize

- **Color**: Change material color
- **Transform**: Move, scale, rotate
- **Delete**: Remove objects

### Step 6: Export

Click **"Export Scene"** to save your work as JSON.

## 🎮 Controls

- **Rotate View**: Left-click + drag
- **Pan View**: Right-click + drag
- **Zoom**: Scroll wheel
- **Select**: Click object
- **Delete**: Click × in objects list

## 📝 Workflow Example

### Creating a Scene:

1. **Add Floor**:
   - Add Plane primitive
   - Scale: [10, 1, 10]
   - Position: [0, 0, 0]

2. **Add House**:
   - Load "Modern Home" sample
   - Position on floor

3. **Add Car**:
   - Load "Sports Car" sample
   - Position near house

4. **Add Details**:
   - Add primitives for trees, etc.
   - Customize colors

5. **Export**:
   - Save your scene

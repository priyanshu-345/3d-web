# 3D Design Editor Guide

## 🎨 Overview

The 3D Design Editor is a web-based tool that allows you to create basic 3D interior designs directly in your browser. It's like a simplified version of Blender, but accessible through the web!

## 🚀 How to Access

1. Navigate to **"Design Editor"** from the main navigation menu
2. Or go directly to: `http://localhost:5173/editor`

## ✨ Features

### Add Shapes
- **Cube**: Perfect for furniture, boxes, tables
- **Sphere**: Great for decorative items, lamps
- **Cylinder**: Ideal for columns, vases, containers
- **Plane**: Use for floors, walls, surfaces

### Edit Objects
- **Position**: Move objects in X, Y, Z directions
- **Scale**: Resize objects (make bigger or smaller)
- **Rotation**: Rotate objects on any axis
- **Color**: Choose from 12 different colors

### View Controls
- **Rotate**: Left-click and drag to rotate the view
- **Pan**: Right-click and drag to move the view
- **Zoom**: Scroll wheel to zoom in/out
- **Touch**: Works on mobile devices too!

## 📝 Step-by-Step Guide

### Creating Your First Design

1. **Add a Floor**
   - Click "+ Add Shape"
   - Select "Plane (Floor/Wall)"
   - In Properties panel, set:
     - Scale: X=5, Y=1, Z=5 (makes it bigger)
     - Position: Y=-0.5 (lowers it to ground level)
   - Change color to your preferred floor color

2. **Add Furniture**
   - Click "+ Add Shape"
   - Select "Cube" for a table
   - Set Position: X=0, Y=0.5, Z=0
   - Set Scale: X=2, Y=0.3, Z=1
   - Choose a color

3. **Add Decorative Items**
   - Add a "Sphere" for a lamp
   - Position it on the table
   - Scale it appropriately
   - Choose a color

4. **Arrange Everything**
   - Click on objects to select them
   - Adjust position, scale, and rotation
   - Use the color picker to change colors

5. **Export Your Design**
   - Click "Export Model" button
   - Your design will be saved as JSON
   - You can later upload it to the gallery

## 💡 Tips & Tricks

1. **Grid Helper**: The grid helps you align objects
2. **Axes Helper**: Red=X, Green=Y, Blue=Z
3. **Selection**: Selected objects glow slightly
4. **Delete**: Click the × button in the objects list
5. **Multiple Objects**: Add as many shapes as you want!

## 🔄 Workflow

1. **Plan**: Think about what you want to create
2. **Build**: Add shapes and arrange them
3. **Refine**: Adjust positions, sizes, colors
4. **Export**: Save your design
5. **Upload**: Use admin panel to upload to gallery

## 📱 Mobile Usage

The editor works on mobile devices too!
- Touch and drag to rotate
- Pinch to zoom
- Tap objects to select

## ⚠️ Limitations

- This is a basic editor for simple designs
- For complex 3D models, use Blender desktop app
- Export creates JSON file (not .glb)
- For production-quality models, create .glb files in Blender

## 🆘 Troubleshooting

**Objects not visible?**
- Check if they're positioned correctly
- Make sure scale is not 0

**Can't select objects?**
- Click directly on the object
- Check if it's in the objects list

**Export not working?**
- Make sure you have at least one object
- Check browser console for errors

# 🏠 Interior Design Studio - Complete Guide

## Overview

The **Interior Design Studio** is a professional 3D interior design tool that allows you to create, customize, and visualize interior spaces with realistic materials, lighting, and furniture placement.

- **Ceiling Colors**: Customizable ceiling

## 🚀 Step-by-Step Tutorial

### Step 1: Access the Studio
1. Go to **"🏠 Design Studio"** in navigation
2. Or visit: `http://localhost:5173/interior-studio`

### Step 2: Set Up Your Room

1. **Click "Room" Tab**
2. **Set Dimensions**:
   - Width: 8m (default)
   - Height: 3m (default)
   - Depth: 6m (default)
3. **Choose Colors**:
   - Wall: White (#ffffff)
   - Floor: Brown (#8b7355)
   - Ceiling: Light gray (#f5f5f5)

### Step 3: Add Furniture

1. **Click "Furniture" Tab**
2. **Add Items**:
   - Click on furniture icons to add
   - Items appear in center of room
3. **Place Furniture**:
   - Click on furniture to select
   - Use Properties panel to adjust position
   - Drag or use X, Y, Z inputs

### Step 4: Customize Materials

1. **Click "Materials" Tab**
2. **Choose Preset**:
   - Select material type (Wood, Fabric, etc.)
3. **Adjust Properties**:
   - Change color
   - Adjust roughness
   - Adjust metalness
4. **Apply to Furniture**:
   - Select furniture first
   - Changes apply automatically

### Step 5: Set Up Lighting

1. **Click "Lighting" Tab**
2. **Choose Preset**:
   - Daylight for bright scenes
   - Evening for warm glow
   - Night for dramatic lighting
3. **Fine-tune**:
   - Adjust ambient intensity
   - Adjust directional intensity
   - Enable/disable shadows

### Step 6: Adjust Camera

1. **Click "Camera" Tab**
2. **Choose Angle**:
   - Perspective for normal view
   - Top for floor plan view
   - Front/Side for elevation
3. **Customize Position**:
   - Adjust X, Y, Z values
   - Toggle grid and axes

### Step 7: Animate & Export

1. **Start Animation**:
   - Click "▶️ Animate" button
   - Watch your design come alive
2. **Capture Screenshot**:
   - Click "📸 Screenshot"
3. **Export Design**:
   - Click "💾 Export Design"
   - Save as JSON file

## 🎨 Design Tips

### Creating a Living Room

1. **Room Setup**:
   - Dimensions: 8m x 3m x 6m
   - Wall: Light beige
   - Floor: Dark wood
   - Ceiling: White

2. **Add Furniture**:
   - Sofa (center, facing TV)
   - Table (in front of sofa)
   - TV (on wall)
   - 2 Chairs (sides)
   - Lamp (corner)
   - Plant (corner)

3. **Materials**:
   - Sofa: Fabric material
   - Table: Wood material
   - TV: Metallic material

4. **Lighting**:
   - Preset: Daylight
   - Shadows: Enabled

### Creating a Bedroom

1. **Room Setup**:
   - Dimensions: 6m x 3m x 5m
   - Wall: Soft blue
   - Floor: Light wood
   - Ceiling: White

2. **Add Furniture**:
   - Bed (center)
   - Cabinet (side wall)
   - 2 Lamps (bedside)
   - Plant (corner)

3. **Materials**:
   - Bed: Fabric
   - Cabinet: Wood
   - Lamps: Metallic

4. **Lighting**:
   - Preset: Warm
   - Ambient: 0.5
   - Directional: 0.8

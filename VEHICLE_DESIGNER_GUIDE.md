## Overview

Transform your project into a professional **Vehicle 3D Design Studio**! Create cars, bikes, trucks, and any vehicle you can imagine using our comprehensive 3D modeling tools.

## 🚀 How to Use

### Step 1: Access the Editor
1. Go to **"🚗 Car Designer"** in the navigation menu
2. Or visit: `http://localhost:5173/vehicle-editor`

### Step 2: Build Your Vehicle

#### Basic Car Structure:
1. **Add Car Body**
   - Click "Body" category
   - Add "Car Body"
   - Position: [0, 0.6, 0]
   - Scale: [1, 1, 1]

2. **Add Wheels**
   - Click "Wheels" category
   - Add 4 "Wheel" parts
   - Front Left: Position [-1.2, 0.2, 0.8]
   - Front Right: Position [1.2, 0.2, 0.8]
   - Rear Left: Position [-1.2, 0.2, -0.8]
   - Rear Right: Position [1.2, 0.2, -0.8]

3. **Add Lights**
   - Click "Lights" category
   - Add "Headlight" (2x for front)
   - Add "Taillight" (2x for rear)
   - Position on front and rear

4. **Add Details**
   - Add "Grille" on front
   - Add "Mirror" on sides
   - Add "Windshield" and "Rear Window"

### Step 3: Customize

1. **Select Parts**: Click on any part to select
2. **Edit Properties**:
   - **Position**: Move parts (X, Y, Z)
   - **Scale**: Resize parts
   - **Rotation**: Rotate parts
   - **Color**: Change paint color

3. **Color Options**: 14 different colors available

### Step 4: Export

Click **"Export Design"** to save your vehicle model as JSON file.

- **Zoom**: Scroll wheel
- **Select Part**: Click on part
- **Delete Part**: Click × in parts list

## 📐 Advanced Techniques

### Positioning Tips
- **X-axis**: Left (-) to Right (+)
- **Y-axis**: Down (-) to Up (+)
- **Z-axis**: Back (-) to Front (+)

### Scaling
- Scale [1, 1, 1] = Original size
- Scale [2, 2, 2] = Double size
- Scale [0.5, 0.5, 0.5] = Half size

### Rotation
- Values in degrees
- 0° = No rotation
- 90° = Quarter turn
- 180° = Half turn
